# box-smashing-game

This is a game where you dash into boxes.

Answers To Questions That Might Be Asked:
The game doesn't go fullscreen because if I change the resolution from 1080p then everything breaks.
I'm gonna update it more.
Pressing Numpad keys changes the stage because it's an old debug feature and I forgot to take it out before uploading this.
The enemy selection system is ripped straight from Nullscape. I thought it was a cool concept.
No I will not change the graphics. I know it looks like a Scratch game.
The reason the game doesn't let you go past Level 12 is because I haven't added more stages yet. It wouldn't take any effort, no.

Enemies and Tips:
Sentry - Keeps its distance and fires bullets. It also does contact damage, so don't try to touch it.
Snail - Be careful when hitting boxes near one. There's always a chance you get ricocheted directly into its loving embrace.
Mirror - Honestly, don't pick this one. Unless you want to see the cool conga line that they form when you have multiple.
Bouncer - Sometimes they can get stuck in the corner and not move at all. I still haven't fixed that. It takes 8 and 1/6th minutes to get to it's max speed.
Guardian - Only spawns past level 5. It can damage lots of boxes at once if you line it up right.
Mimic - Also only spawns past level 5. Although they flash when hit by an enemy, Mimics can only be damaged by player dashes.


That's it. Enjoy.
